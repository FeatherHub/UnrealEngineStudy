    FVertexFactory*       VertexFactory;       // vertex 를 어떻게 조립할지

1. View 구성 — FSceneView 생성. 카메라의 ViewMatrix, ProjectionMatrix, 뷰포트 크기

2. Visibility 계산 — 어떤 primitive 가 frustum 안에 있는가 (-> VisibilityMap)

3. Dynamic mesh elements 수집
  SceneProxy -> GetDynamicMeshElements(Views, ViewFamily, VisibilityMap, Collector)
  // SceneProxy 가 자신의 FMeshBatch 를 Collector 에 Add

4. 렌더 패스 실행
  각 패스가 자기 관심사의 FMeshBatch 들을 처리

5. GPU draw
  VertexFactory type × Material type → 셰이더 permutation 선택
  PSO 바인딩
  Uniform buffer 들 바인딩 (PrimitiveUB, ViewUB, MaterialUB)
  VertexFactory 가 vertex stream 설정

FScene 은 렌더 스레드의 세계 표현
"이 프레임에 무엇을 그릴 것인가" 의 완결된 데이터베이스.


FScene
  ├─ FPrimitiveSceneInfo + SceneProxy
  ├─ FLightSceneInfo + FLightSceneProxy
  ├─ Sky Atmosphere, Height Fog, Reflection Capture 등 환경 요소
  ├─ Static mesh draw list (정적 오브젝트의 사전 빌드된 draw call)
  ├─ GPU scene data
  ├─ Primitive bounds / spatial data
  └─ Renderer가 View별로 순회할 render state
