골짜기의 가장 바닥에 공이 멈춰 있는 상태를 상상해 보세요. 이곳이 바로 **평형점(Equilibrium point)**입니다

이 바닥 점에서는 힘이 균형을 이루어 알짜힘이 0입니다
에너지 그래프로 보면, 이 지점은 곡선의 **가장 낮은 골짜기(최솟값)**에 해당합니다

테일러 전개(Taylor expansion): 복잡한 것을 단순하게 보기

우리가 관심 있는 것은 평형점(골짜기 바닥) 근처에서의 아주 작은 움직임입니다. 
수학자 테일러는 **"아무리 복잡한 곡선이라도 특정 점 근처에서는 
다항식(x, x², x³...)의 합으로 아주 비슷하게 흉내 낼 수 있다"**는 것을 발견했습니다

평형점(x=0) 근처에서 에너지 U(x)를 테일러 전개하면 다음과 같은 형태가 됩니다

U(x) = U(0) + U'(0)x + 1/2 U''(0)x² + ...

첫 번째 항 U(0): 이는 단순히 에너지의 기준점(높이)입니다. 
물리적으로 에너지는 '차이'가 중요하므로 이 값은 0으로 두어도 무방합니다

두 번째 항 U′(0)x: 여기서 U′(0)은 골짜기 바닥에서의 '기울기'를 뜻합니다. 하지만 바닥 점은 평평하므로 기울기가 0입니다. 
따라서 이 항은 사라집니다

결과: 변위 x가 아주 작을 때, 가장 큰 영향을 미치는 항은 결국 ½ U''(0)x²이 됩니다

여기서 U′′(0)(곡률의 정도)을 상수 k라고 부르기로 약속하면, 에너지는 U ≈ 1/2 kx²이라는 단순한 포물선 형태가 됩니다

곡선 위의 한 점에서, 아주 작은 변화만 보면, 입력의 변화와 출력의 변화가 비례한다
이 비례 상수가 **미분계수(derivative)**다.

---

f(x) = 3x
x가 1 늘면 f는 3 늘고, x가 0.01 늘면 f는 0.03 늘고, 
x가 Δx 늘면 f는 3Δx 는다

Δf/Δx = 3Δx/Δx

여기서 Δx를 아무리 크게 잡든 작게 잡든 비율은 3이다
어디서 보든, 얼마나 넓게 보든, 비율이 같다

f(x) = x²

x = 2에서 x = 4    로 갈 때: Δf/Δx = (16       - 4) / (2)      = 12 / 2           = 6
x = 2에서 x = 3    로 갈 때: Δf/Δx = (9        - 4) / (1)      = 5 / 1            = 5
x = 2에서 x = 2.1  로 갈 때: Δf/Δx = (4.41     - 4) / (0.1)    = 0.41 / 0.1       = 4.1
x = 2에서 x = 2.01 로 갈 때: Δf/Δx = (4.0401   - 4) / (0.01)   = 0.0401 / 0.01    = 4.01
x = 2에서 x = 2.001로 갈 때: Δf/Δx = (4.004001 - 4) / (0.001)  = 0.004001 / 0.001 = 4.001

비율이 변한다. 하지만 Δx를 줄일수록 비율이 어떤 값에 수렴한다
도착 지점을 출발 지점에 한없이 가까이 가져가면, 비율이 한없이 4에 가까워진다
Δx가 작을수록 근사가 정확해진다

이 수렴하는 값 4가 x = 2에서의 미분계수다

---

"x = 2 근처에서, x가 아주 조금 변하면 f는 그 4배만큼 변한다."

B를 A에 가까이 가져간다. 점점 가까이. 결국 B가 A와 겹친다
두 점이 겹치면 Δf = 0, Δx = 0. 비율은 0/0이다

미분은 이 축퇴에서 살아남는 정보를 추출하는 것이다

A = 2, 4,
B = 2 + Δx, (2 + Δx)²

Δf/Δx를 전개하면

Δf = (2 + Δx)² - 4 = 4 + 4Δx + Δx² - 4 = 4Δx + Δx²
Δf/Δx = 4 + Δx

Δf/Δx 안에 두 종류의 항이 있다

4: Δx에 의존하지 않는다. 두 점이 아무리 가까워져도, 아무리 멀어져도, 이 항은 변하지 않는다
Δx: 두 점의 간격에 비례한다. B가 A에 겹쳐지면 0이 된다

미분계수 f'(2) = 4는, 두 점이 겹쳐질 때 살아남는 항이다
0/0이라는 부정형 안에서, 간격에 의존하지 않는 부분만 추출한 것이다.

---

Δx는 측정의 부산물이다. 두 점의 간격을 얼마로 잡았느냐에 따라 달라진다. 이것은 곡선의 성질이 아니라 관찰자의 선택이다.
