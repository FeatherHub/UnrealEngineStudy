          }
      // 쓴 내용을 FReplicationRecord 에 적재 (ack/재전송용)
    }

             : 전역에서 긁어와 FNetHandle→내부 i 번역, 자기 dirty bitset[i] set

push의 커버리지(trait)

HasFullPushBasedDirtiness = 그 오브젝트의 모든 복제 프로퍼티가 property-level push로 덮임
changemask가 push만으로 완성 → poll 아예 생략

HasPartialPushBasedDirtiness = 일부만 push로 덮임
덮인 건 push가 changemask로 알려주고, 나머지는 여전히 poll.

FProperty ReplicationState::Poll PropertyReplicationState (Owner)
{
    for (MemberIndex)
    {
        Property:FProperty* = Descriptor::MemberProperties[MemberIndex];

        UObjectValue =
            reinterpret_cast<uint8*>(Owner)
            + Property->GetOffset_ForGC()
            + Property->GetElementSize() * ArrayIndex;

        ExternalValue =
            StateBuffer
            + Descriptor::MemberDescriptors[MemberIndex].ExternalMemberOffset;

        Property 값과 ExternalValue를 비교
        변경되었으면 ChangeMask를 표시
        Property 연산으로 UObjectValue를 ExternalValue에 복사
    }
}
