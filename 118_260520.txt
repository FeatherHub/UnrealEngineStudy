GameInstance 

During a level transition, the World and everything in it gets destroyed, 
so anything crossing that boundary must live outside it

a container for whatever needs to survive level transitions
which varies by game type

GameInstance holds ULocalPlayer objects ; ULocalPlayer holds ULocalPlayerSubsystem
GameInstance holds GameInstanceSubsystem

In an RPG that's inventory and quest state, 
in an FPS it's settings and score (오늘 몇 판 했는지, 세션 킬 수)
in online games it's network and matchmaking state
맵 전환 때 어떤 정보/에셋을 보여줄지

for anything whose lifetime should match the game process
