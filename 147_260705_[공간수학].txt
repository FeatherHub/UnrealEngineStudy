기준축 e₁ = (1,0)을 θ만큼 돌리면 단위원 위 어딘가에 앉아. 
그 점의 두 좌표를 그냥 C(θ), S(θ)라 부르자.

C, S는 정체 모를 두 함수고, 아는 건 C² + S² = 1

e₂ = (0,1)은 e₁을 90° 돌린 것이야. 그리고 2D 회전끼리는 교환해

R(θ)e₂ = R(θ)(J e₁) = J(R(θ)e₁) = J(C, S) = (−S, C)

[ C  −S ]
[ S   C ]

---

R(α+β) = R(α)R(β)

이걸 행렬 곱으로 적고 성분을 읽기만 하면 돼.

[ C(α) −S(α) ] [ C(β) −S(β) ]
[ S(α)  C(α) ] [ S(β)  C(β) ]

왼쪽 열을 곱하면

위 = C(α)C(β) − S(α)S(β)
아래 = S(α)C(β) + C(α)S(β)

그런데 이 왼쪽 열은 R(α+β)의 첫 열, 곧 (C(α+β), S(α+β))와 같아야 해

C(α+β) = C(α)C(β) − S(α)S(β)
S(α+β) = S(α)C(β) + C(α)S(β)
