FTaskBase
├─ 자물쇠 개수        아직 몇 개를 기다려야 하나
├─ 뒷면 명단          내가 끝나면 풀어줄 쪽지들 + 도장 여부
├─ 앞 일 목록         나를 막고 있는 쪽지들 (거슬러 올라갈 때 씀)
├─ 붙잡은 사람 수     0 이 되면 지워짐
├─ 소속 pipe          한 줄 서기에 속해 있다면 어느 줄
├─ 지금 실행 중인 손  어느 thread 가 지금 이걸 돌리고 있나 // ThreadId; OS 가 모든 thread 에 매겨주는 정수
├─ 깨우는 장치        기다리는 사람에게 알릴 때 씀
├─ 할 일 내용         실제 lambda
├─ 결과 놓을 자리     결과를 반환하는 쪽지라면
└─ 손잡이

├─ 부를 함수 하나     내용: "큰 쪽지에게 돌아가서 실행하라고 해"

└─ 여분 포인터 하나
